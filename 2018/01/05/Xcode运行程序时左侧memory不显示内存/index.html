Xcode运行程序时左侧memory不显示内存

发表于 2018-01-05 | 更新于 2018-03-06 | 分类于 iOS开发

运行程序后，xcode不显示当前使用的内存情况

问题是打开了僵尸对象跟踪（enable zoombie Objects），将其关闭即可

打开 product -> SCheme -> EditSCheme -> Diagnostics -> enable zoombie Objects 取消选中

就可以继续显示了
