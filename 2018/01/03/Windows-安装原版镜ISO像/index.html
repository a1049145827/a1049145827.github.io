Windows 安装原版镜ISO像

发表于 2018-01-03 | 分类于 Windows安装

安装程序无法创建新的系统分区，也无法定位现有系统分区

最近同事在闲鱼淘了个笔记本来玩，安装系统的时候发现主硬盘无法引导启动，用原版镜像安装的时候报错无法继续执行。

报错问题：安装程序无法创建新的系统分区，也无法定位现有系统分区

最终只能做个PE U盘来帮忙，顺便记录下这个烦人的问题。

解决步骤：

  1. 第一步：把win7或win8、win10镜像发在你电脑的非系统盘的其他硬盘上，或者放在U盘也可以。

  2. 第二步：重启机器，通过U 盘启动．进入win pe系统，关于这点我说一下，有些朋友也许不知道什么叫win pe系统，这个win pe 究竟有什么用途，其实你可以把win pe 理解为一个微缩版本win xp，他和win xp 使用差不多。只是大部分的功能被禁用了，所以他的体积也非常小，虽然大部分的功能被禁用，可是他不影响第三方软件使用，比如Ghost , PQ 等一些软件依旧在win pe运行自如。

  3. 第三步：如果PE里带虚拟光驱，此步骤可以忽略。（把windows 7的ISO镜像解压到电脑的非系统盘的其他硬盘上。如D:\windows.…。当然你也可以不用建立 win7 文件夹。在这里再说明一点就是，一定要把你想分的分区分好．尽量不要频繁调整硬盘各个分区的大小，尤其是主分区，一定要考虑周全。）

  4. 第四步：进入win pe系统，格式化C盘为NTFS。C盘最好有15 G以上的空间，安装完 win7 大概要用 8G 。我在这里建议大家分 50G 。

  5. 第五步（最关键一步）：从你解压的安装程序的文件夹中找到boot, bootmgr 和 sources 这三个文件或文件夹，并且复制到C盘根目录下。

  6. 第六步：在win pe系统中运行cmd ，输入“ c:\boot\bootsect.exe /nt60 c: ”（注意exe和/nt60、/nt60和c:之间有空格），然后回车，看到提示successful 字样（画面会很快闪过，以至于来不及看。）的语句即成功了！然后重起电脑。

  7. 第七步：可以把U盘从电脑中拔出，然后等待系统从硬盘启动。或者加载U盘启动项后选择从本地引导启动。
