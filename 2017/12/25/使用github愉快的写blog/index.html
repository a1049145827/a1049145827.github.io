添加图片

因为Markdown里面只能写文字，所以添加图片是采用链接的方式，我们就需要先将图片传到网上，然后获取链接才能进行展示。
上传图片的方式有很多，这里我们用github上传的方式，正好blog也在github上面，方便管理。

在github新建一个仓库，通过git将图片上传至仓库。在github页面打开该仓库，找到图片对应的元素，点击预览，然后右键复制链接即可拿到图像地址。如下图：
